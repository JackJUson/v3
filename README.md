## ðŸ›  Installation & Set Up

1. Get the project

   ```sh
   git clone git@github.com:JackJUson/v3.git
   ```

2. Install and use the correct version of Node using [NVM](https://github.com/nvm-sh/nvm)

   ```sh
   nvm install
   ```

3. Install dependencies

   ```sh
   npm install
   ```

4. Start the development server

   ```sh
   npm start
   ```
